# SVG Generator

  ## Description
  
  A Node.js command-line application that takes in user input to generate a logo and save it as an SVG file. User can enter up to three characters for the text, pick the shape of the logo, and choose the color of both text and shape.

  Link to walkthrough video: https://drive.google.com/file/d/1N49-DD6wbemsHIsi5r23lI1llKeYzP2g/view
